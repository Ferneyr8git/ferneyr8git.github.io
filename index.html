Párrafo 1

Las aplicaciones web son programas que se ejecutan
directamente en tu navegador, sin necesidad de instalar
software adicional en tu computadora. Esto las hace muy
accesibles y fáciles de usar desde cualquier dispositivo
con conexión a internet. Desde redes sociales hasta tiendas
en línea, las aplicaciones web han revolucionado la forma en
que interactuamos con la información y realizamos tareas cotidianas.

Comentario 1.1

cita de Google

Las aplicaciones web funcionan de manera similar a una conversación.
Tú (el usuario) envías una solicitud a un servidor remoto
(como hacer clic en un botón), el servidor procesa esa solicitud y
te devuelve una respuesta (por ejemplo, mostrando una nueva página).
Este intercambio de información ocurre en tiempo real, lo que te permite
interactuar con la aplicación de forma fluida.

Comentario 1.2

Como crear páginas web

Las aplicaciones web se construyen utilizando una combinación de lenguajes
de programación como HTML, CSS y JavaScript. HTML estructura el contenido,
CSS le da estilo y JavaScript agrega interactividad. Estos lenguajes se combinan
con frameworks y bibliotecas que facilitan el desarrollo y permiten crear interfaces
de usuario más complejas y dinámicas.
